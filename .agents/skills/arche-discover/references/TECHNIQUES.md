# Brainstorming Techniques

61 creativity techniques across 10 categories.

## Arche-leveraged techniques

These techniques shine when prior Arche context has been loaded in Phase 1 — they explicitly draw on existing entity/concept/query pages. When AI-Recommended mode is in use and substantial context was loaded, prefer these:

- **Analogical Thinking** — parallels with documented patterns
- **Cross-Pollination** — adapt solutions from other domains the Arche covers
- **First Principles Thinking** — strip established claims to fundamentals
- **Reversal / Inversion** — flip an existing concept page
- **Forced Relationships** — bridge two named concepts
- **Concept Blending** — merge two existing Arche concepts into a new one
- **Trait Transfer** — borrow attributes from a documented success
- **Failure Analysis** — mine recorded failures and strikethrough'd claims
- **Constraint Mapping** — list constraints the Arche has captured for the topic
- **Question Storming** — generate questions where the Arche has gaps
- **Reverse Brainstorming** — break existing solutions
- **Five Whys** — drill into documented causes

## Collaborative

| Technique | Description |
| :--- | :--- |
| **Yes And Building** | Build momentum through positive additions where each idea becomes a launching pad. Use prompts like "Yes and we could also..." or "Building on that idea..." to create energetic collaborative flow. |
| **Brain Writing Round Robin** | Silent idea generation followed by building on others' written concepts. Gives quieter voices equal contribution; sequence is write silently → pass ideas → build on received concepts. |
| **Random Stimulation** | Use random words or images as creative catalysts to force unexpected connections. Ask how random elements relate, what connections exist, what relationships can be forced. |
| **Role Playing** | Generate solutions from multiple stakeholder perspectives. Embody each role and ask what they want, how they'd approach the problem, what matters most. Pair with entity pages — embody documented stakeholders. |
| **Ideation Relay Race** | Rapid-fire idea building under time pressure. 30-second additions, quick building, fast passing. Maintains momentum and prevents overthinking. |

## Creative

| Technique | Description |
| :--- | :--- |
| **What If Scenarios** | Explore radical possibilities by questioning all constraints. Prompts: "What if we had unlimited resources?", "What if the opposite were true?", "What if this problem didn't exist?" |
| **Analogical Thinking** | Find solutions by drawing parallels to other domains. Ask "This is like what?", "How is this similar to...?", "What other examples come to mind?" |
| **Reversal / Inversion** | Deliberately flip the problem upside down to reveal hidden assumptions. Ask "What if we did the opposite?", "How could we make this worse?", "What's the reverse approach?" |
| **First Principles Thinking** | Strip away assumptions and rebuild from fundamental truths. Ask "What do we know for certain?", "What are the fundamental truths?", "If we started from scratch?" |
| **Forced Relationships** | Connect unrelated concepts to spark innovative bridges. Take two unrelated things, find connections, identify bridges, explore how they could work together. |
| **Time Shifting** | Explore solutions across different time periods. "How would this work in the past?", "What about 100 years from now?", "What time-based solutions apply?" |
| **Metaphor Mapping** | Use extended metaphors as thinking tools. "This problem is like a [metaphor]" — then extend the metaphor and map elements to discover insights. |
| **Cross-Pollination** | Transfer solutions from completely different industries. "How would industry X solve this?", "What patterns work in field Y?", "How could we adapt solutions from domain Z?" |
| **Concept Blending** | Merge two or more existing concepts to create entirely new categories. Beyond simple combination — what emerges when concepts merge, what new category is created? |
| **Reverse Brainstorming** | Generate problems instead of solutions to identify hidden opportunities. "What could go wrong?", "How could we make this fail?", "What problems could we create?" |
| **Sensory Exploration** | Engage all five senses to discover multi-dimensional solution spaces. What do ideas feel, smell, taste, sound like? |

## Deep

| Technique | Description |
| :--- | :--- |
| **Five Whys** | Drill down through layers of causation to uncover root causes. Ask "Why did this happen?" repeatedly until reaching fundamental drivers. |
| **Morphological Analysis** | Systematically explore all parameter combinations. Identify key parameters, list options for each, try different combinations, identify emerging patterns. |
| **Provocation Technique** | Use deliberately provocative statements as starting points. "What if [provocative statement]?", "How could this be useful?", "Extract the principle." |
| **Assumption Reversal** | Challenge and flip core assumptions to rebuild from new foundations. "What assumptions are we making?", "What if the opposite were true?" |
| **Question Storming** | Generate questions before seeking answers to properly define problem space. Only questions, no answers yet. Focus on what we don't know. |
| **Constraint Mapping** | Identify and visualize all constraints. Which are real vs imagined? How to work around or eliminate each barrier? |
| **Failure Analysis** | Study failures to extract insights and avoid pitfalls. What went wrong, why, what lessons emerged, how to apply failure wisdom? |
| **Emergent Thinking** | Allow solutions to emerge organically without forcing linear progression. What patterns emerge? What wants to happen naturally? |

## Structured

| Technique | Description |
| :--- | :--- |
| **SCAMPER Method** | Seven lenses for methodical innovation: **S**ubstitute, **C**ombine, **A**dapt, **M**odify, **P**ut to other uses, **E**liminate, **R**everse. |
| **Six Thinking Hats** | Six distinct perspectives: White (facts), Red (emotions), Yellow (benefits), Black (risks), Green (creativity), Blue (process). |
| **Mind Mapping** | Visually branch ideas from a central concept. Main idea center, branches out, identify sub-branches and connections. |
| **Resource Constraints** | Generate solutions under extreme limitations. "What if you had only $1?", "No technology?", "One hour to solve?" |
| **Decision Tree Mapping** | Map all possible decision paths and outcomes. Identify paths, decision points, and where different choices lead. |
| **Solution Matrix** | Systematic grid of problem variables × solution approaches. Test combinations, identify most effective pairings. |
| **Trait Transfer** | Borrow attributes from successful solutions in unrelated domains. What traits make success X work? How to transfer them? |

## Introspective / Delight

| Technique | Description |
| :--- | :--- |
| **Inner Child Conference** | Channel childhood curiosity. What would 7-year-old you ask? Use "why why why" questioning. Forbid boring thinking. |
| **Shadow Work Mining** | Explore what you're actively avoiding or resisting. Where's the resistance? What scares you? Mine the shadows for buried wisdom. |
| **Values Archaeology** | Excavate deep personal values driving decisions. What really matters? What's non-negotiable? What core values guide your choices? |
| **Future Self Interview** | Seek wisdom from a wiser future self. What would your 80-year-old self tell younger you? |
| **Body Wisdom Dialogue** | Let physical sensations and gut feelings guide ideation. What does your body say? Where do you feel it? Follow physical cues. |
| **Permission Giving** | Grant explicit permission to think impossible thoughts and break self-imposed barriers. Permission to explore, try, experiment, break free. |

## Theatrical

| Technique | Description |
| :--- | :--- |
| **Time Travel Talk Show** | Interview past/present/future selves for temporal wisdom across life stages. |
| **Alien Anthropologist** | Examine familiar problems through completely foreign eyes. Become an outsider observer; what seems strange? |
| **Dream Fusion Laboratory** | Start with impossible fantasy solutions, then reverse-engineer practical steps. Backwards design from the dream. |
| **Emotion Orchestra** | Let different emotions lead separate brainstorming passes — angry, joyful, fearful, hopeful — then harmonize. |
| **Parallel Universe Cafe** | Explore solutions under alternative reality rules. Different physics, different social norms, changed history. |
| **Persona Journey** | Embody different archetypes or personas. Become the archetype; what would they see that normal thinking misses? |

## Wild

| Technique | Description |
| :--- | :--- |
| **Chaos Engineering** | Deliberately break things to discover robust solutions. Stress-test ideas against worst-case scenarios. |
| **Guerrilla Gardening Ideas** | Plant unexpected solutions in unlikely places. Surprise and unconventional placement for stealth innovation. |
| **Pirate Code Brainstorm** | Take what works from anywhere and remix without permission. Rule-bending, rapid prototyping, maverick thinking. |
| **Zombie Apocalypse Planning** | Design solutions for extreme survival scenarios. Strip away all but essential functions to find core value. |
| **Drunk History Retelling** | Explain complex ideas with uninhibited simplicity. Remove overthinking; simplify to absurdity. |
| **Anti-Solution** | Generate ways to make the problem worse. Reveals hidden assumptions through destructive creativity. |
| **Quantum Superposition** | Hold multiple contradictory solutions simultaneously until best emerges through observation. |
| **Elemental Forces** | Sculpt solutions with natural elements. How would earth, fire, water, air shape this? |
