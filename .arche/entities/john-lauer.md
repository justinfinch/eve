# John Lauer

Founder of [Adom Industries](adom-industries.md), who laid out the company's thesis in an Aug 2025 Dallas Innovates interview ([source](../sources/adom-decoded-and-poc-plan.md)).

## Facts

- Self-funded Adom with ~$10M of his own capital ([source](../sources/adom-decoded-and-poc-plan.md)).
- Framed Adom as "a data center, but for atoms, not bits" / "compute for atoms" / "the Amazon Web Services of electronics prototyping," attacking the problem that "what takes minutes in software can take months in hardware" ([source](../sources/adom-decoded-and-poc-plan.md)).
- Career throughline is a repeated pattern: put a hard-to-reach physical resource behind a clean software API and sell it as a utility — SMS carriers → Zipwhip's toll-free texting (reliability/uptime DNA, "99.999%") → CNC-over-browser with ChiliPeppr → now the whole electronics lab ([source](../sources/adom-decoded-and-poc-plan.md)).
- Pioneered the browser-to-serial bridge pattern via Serial Port JSON Server (SPJS) and ChiliPeppr — the lineage behind Adom's "Hydrogen Desktop" local-agent client and the Web Serial path usable in a POC ([source](../sources/adom-decoded-and-poc-plan.md)).

## Relationships

- Founder/funder of [Adom Industries](adom-industries.md).
- His SPJS/ChiliPeppr browser-to-serial lineage underpins the host-bridge layer in [Adom Technical Architecture](../concepts/adom-technical-architecture.md) and the recommended [POC](../concepts/poc-mini-molecule-cloud-workbench.md).
